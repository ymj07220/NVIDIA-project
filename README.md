# 🌌 CosmicWeb-AI (KISTI-NVIDIA GPU Hackathon)

> **우주 거대 구조(Cosmic Web) 분석을 위한 GNN 기반 암흑물질 할로 질량 예측 프로젝트**

## 🎯 프로젝트 목표
- **IllustrisTNG** 시뮬레이션 데이터를 활용한 우주 구조 그래프 모델링
- **GNN(Graph Neural Networks)**을 통한 암흑물질 질량 추론 가속화
- NVIDIA GPU 최적화를 통한 전통적 시뮬레이션 대비 성능 혁신

## 🛠 기술 스택
- **Language:** Python 3.9+
- **Framework:** PyTorch, PyTorch Geometric
- **Hardware:** NVIDIA GPU (CUDA 가속)
- **Tools:** h5py, CuPy, NVIDIA Nsight

## 👥 팀원 및 역할
- **이훈행 (Physics):** 데이터 전처리, 물리적 검증, GPU 성능 프로파일링
- **심재형 (AI):** GNN 모델 설계 및 하이퍼파라미터 최적화
